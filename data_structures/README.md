# Data Structures

This section contains implementations and explanations of various data structures.

## Available Data Structures

1. Linked List
   - Singly Linked List
   - Doubly Linked List
   - Circular Linked List
2. Stack
3. Queue
   - Simple Queue
   - Priority Queue
   - Circular Queue
4. Tree
   - Binary Tree
   - Binary Search Tree
   - AVL Tree
   - Red-Black Tree
5. Heap
   - Min Heap
   - Max Heap
6. Hash Table
7. Graph
   - Adjacency List
   - Adjacency Matrix

## Structure

Each data structure is organized in its own directory with the following structure:

```
data_structure_name/
├── README.md         # Explanation and analysis
├── implementation/   # Implementation in different languages
└── examples/         # Example use cases
```
